# Proxy ARP 模式

高级模式，让 VPN 客户端获得和局域网同网段的 IP。

## 特点

- VPN 客户端使用局域网同网段地址，例如 `192.168.0.240/28`。
- 不做 MASQUERADE/NAT。
- 宿主机开启 Proxy ARP，让局域网设备可以找到 VPN 客户端 IP。
- 需要避开主路由 DHCP 地址池，避免 IP 冲突。

## 示例 `.env`

```env
VPN_MODE=proxyarp
VPN_POOL=192.168.0.240/28
LAN_SUBNET=192.168.0.0/24
# VPN_LOCAL_TS 默认等于 LAN_SUBNET
```

## 安全要求

- 不要使用整个 `192.168.0.0/24` 作为 VPN_POOL。
- 推荐只留小段，例如 `192.168.0.240/28`。
- 必须在主路由 DHCP 里排除这段地址。
- 如果客户端所在地网络也是 `192.168.0.0/24`，可能发生路由冲突。

## 适合场景

- 需要 VPN 客户端在内网里显示为同网段 IP。
- 需要某些只接受同网段访问的服务。
- 能够控制主路由 DHCP 地址池。
