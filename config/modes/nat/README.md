# NAT 模式

默认模式，推荐普通用户使用。

## 特点

- VPN 客户端使用独立网段，例如 `10.66.0.0/24`。
- 容器自动为 VPN 网段做 MASQUERADE/NAT。
- 最稳定，和局域网 DHCP 不冲突。
- 不需要上级路由器增加静态路由。

## 示例 `.env`

```env
VPN_MODE=nat
VPN_POOL=10.66.0.0/24
LAN_SUBNET=192.168.0.0/24
# VPN_LOCAL_TS=0.0.0.0/0
```

## 适合场景

- 手机/电脑远程访问内网。
- VPN 客户端需要上网。
- 不想改主路由配置。
